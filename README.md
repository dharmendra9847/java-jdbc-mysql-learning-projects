# Java JDBC MySQL Learning Projects

This repository contains multiple Java JDBC examples and mini projects demonstrating how to connect Java applications with MySQL databases and perform CRUD operations.

It is designed for students and backend developers who want to learn **JDBC, SQL, and database interaction using Java**.

---

## Technologies Used

- Java
- JDBC (Java Database Connectivity)
- MySQL
- SQL
- IntelliJ IDEA

---

## Project Modules

### 1. JDBC Connection with MySQL
Demonstrates how to establish a connection between a Java application and MySQL database using JDBC.

Features:
- Load JDBC Driver
- Create Connection
- Execute SQL Queries

---

### 2. File IO vs JDBC
Compares storing data using:
- File System
- Database (MySQL)

Helps understand why databases are used in real applications.

---

### 3. JDBC Statement Interface
Example project using `Statement` interface to execute SQL queries.

Includes:
- Insert
- Update
- Delete
- Select

---

### 4. SQL Injection Demonstration
Shows how SQL injection occurs when queries are not written securely.

Important for understanding:
- Security risks
- Why `PreparedStatement` should be used

---

### 5. CRUD Operations using JDBC
A mini project demonstrating:

- Create
- Read
- Update
- Delete

operations using MySQL database.
